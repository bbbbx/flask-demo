# flask-demo

Flask 不是辣椒，是一个角状的容器，和 Bottle 有 PY 交易（同样是以容器命名的 Web 框架）。

- UI：flask-bootstrap
- 数据库：flask-sqlalchemy
- 邮件：flask-mail
    - 异步任务队列：Celery + Redis + RabbitMQ
- 表单：flask-wtf
    - 文件上传：flask-wtf 的 `FileField`、flask-uploads
    - 富文本编辑器：flask-CKEditor
- 日期和时间：flask-moment
- 环境变量：python-dotenv
- HTML 模板引擎：Jinja2
